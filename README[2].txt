CodexIntern Project Package (Tasks 6 & 7)
======================================

Files included:
- gemini_integration.py      : Template code for Google Gemini + Search integration
- speech_to_image.py         : Template code for Speech-to-Image (requires API & network)
- output_image_placeholder.png : Local placeholder image for prompt 'A futuristic flying car in neon city'
- CodexIntern_Task6_7_Report.pdf : This report
- original_assignment_screenshot.jpg : The uploaded assignment image (for reference)

How to run:
1) Open each .py file and add your API keys.
2) Install dependencies: pip install google-generativeai requests speechrecognition pyaudio pillow
3) For Task 7, allow microphone access and run speech_to_image.py, then modify to call the image API.
